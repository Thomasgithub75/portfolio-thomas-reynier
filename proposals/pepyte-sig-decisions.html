Décision 03 — Logique inversée

Haute Moyenne Basse
28 % → Pipeline à alimenter Haute
62 % → Process en cours Moyenne
85 % → Laisser l'inertie Basse

Règle : Priorité décrit une action attendue, pas un état. Haute = besoin d'agir, pas "avancement élevé".

Décision 02 — Plafonné à 90 %

90 % maximum · jamais 100 %
max 90 %

Pourquoi : Tant qu'un contrat n'est pas signé, il reste une incertitude réelle. 100 % donnerait une fausse certitude — et risque de démobiliser le HH sur une offre pas encore close.

Décision 05 — Wording métier

✓ 28 % de signer
✗ Score de conversion
✗ Indice de confiance
✗ Taux de transformation

"Signer" est le verbe du quotidien des headhunters. Un placement = une signature. Pas de jargon — un mot compris sans explication.
