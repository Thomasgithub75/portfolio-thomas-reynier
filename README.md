# Portfolio — Thomas Reynier

Portfolio personnel de Thomas Reynier, Product Designer basé à Paris.

## Structure

```
/
├── index.html          # Page d'accueil (hero, compétences, projets, contact)
├── case-pepyte.html    # Étude de cas Pepyte — UX Research & Matching
├── case-weborama.html  # Étude de cas Weborama — Design System
├── case-nectar.html    # Étude de cas Nectar — Founding Design IA
└── README.md
```

## Stack

- HTML / CSS vanilla (aucune dépendance)
- Police : [Outfit](https://fonts.google.com/specimen/Outfit) via Google Fonts
- Design system : blanc `#FFFFFF`, bleu `#1A56DB`, Outfit 300–600

## Déploiement

Ce site est prévu pour être déployé sur **Vercel** ou **GitHub Pages**.

### Vercel (recommandé)
1. Importer le repo depuis l'interface Vercel
2. Framework preset : **Other**
3. Le fichier `index.html` est automatiquement détecté

### GitHub Pages
Activer Pages dans les paramètres du repo → branche `main` → dossier `/` (root).

## Développement local

Ouvrir directement `index.html` dans un navigateur, ou utiliser un serveur local :

```bash
npx serve .
```
